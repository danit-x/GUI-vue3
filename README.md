# VYBE — Lifestyle & Tech Store

VYBE is a data-driven single page application built with Vue 3, TypeScript, and Tailwind CSS.

## Features

- Product browsing
- Product detail pages
- Authentication simulation
- Shopping cart
- Wishlist
- Search and filtering
- Dark mode
- Responsive design

## Tech Stack

Vue 3  
TypeScript  
Vite  
Tailwind CSS  
Pinia  
Vue Router  
DummyJSON API

## Setup

bun install
bun run dev
